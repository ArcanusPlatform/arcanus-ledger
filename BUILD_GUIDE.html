Arcanus Ledger Build Guide

This project intentionally produces only Android APK, macOS, Windows, and Linux release artifacts.

Allowed Artifacts

Target Build Command Artifact
Android APK flutter build apk --release build/app/outputs/flutter-apk/app-release.apk
macOS flutter build macos --release arcanus-ledger-macos.zip
Windows flutter build windows --release arcanus-ledger-windows.zip
Linux flutter build linux --release arcanus-ledger-linux.tar.gz

Local Build Script

Run the project build script from the repository root:

./build.sh

The script defaults to Android plus the desktop artifact supported by the current host: macOS on macOS, Windows on Windows, Linux on Linux. To choose targets explicitly, set BUILD_TARGETS to a comma-separated subset of android,macos,windows,linux.

The script creates portable storage only under Arcanus Ledger/Data, with SQLite stored in Arcanus Ledger/Data/database/recordkeep_db.sqlite.

GitHub Release Builds

Push a version tag to run the release workflow:

git tag v1.0.0
git push origin v1.0.0

The workflow uploads only arcanus-ledger-android-apk, arcanus-ledger-macos, arcanus-ledger-windows, and arcanus-ledger-linux.
